# 🚨 Fake Internship Detector

A Python-based web application that detects potentially fake internship postings using keyword analysis and rule-based filtering.

## 📌 Project Overview

Fake Internship Detector helps students identify suspicious or fraudulent internship offers by analyzing job descriptions and detecting scam-related keywords.

The system evaluates internship descriptions based on predefined scam indicators and flags postings as:

- ✅ Likely Genuine
- ⚠️ Potentially Fake

## 🛠️ Tech Stack

- Python
- Flask
- HTML / CSS
- CSV (Keyword Dataset)

## 🔍 Features

- Keyword-based scam detection
- Simple and user-friendly web interface
- Fast text analysis
- Rule-based filtering system
- Easy to expand with ML in future
